# GestionAutoFichier

## Utiliser le script

#### Créer un dossier pour le jour même

```bash
./fichierAuto.sh
```

#### Créer un dossier pour un jour donnée

```bash
./fichierAuto.sh jour mois annee
```

#### Supprimer le dossier créer pour le jour même

```bash
./fichierAuto.sh -d
```

OU

```bash
./fichierAuto.sh --delete
```
